# Getting Started with MHU Helpline

This guide will help you get started with the MHU Helpline system as an agent or supervisor.

## System Requirements

Before using the MHU Helpline system, ensure your workstation meets these requirements:

- **Operating System**: Windows 10/11 or macOS 10.15+
- **Browser**: Chrome 80+, Firefox 75+, or Edge 80+
- **Internet Connection**: Minimum 2 Mbps download/upload speed
- **Audio**: Working headset with microphone
- **Optional**: Webcam for video calls (if applicable)

## Logging In

1. Open your web browser and navigate to the MHU Helpline portal
2. Enter your username and password
3. Click "Log In"

<!-- ![Login Screen](/img/login-screen.png) -->

## The Agent Interface

After logging in, you'll see the main agent interface with these key areas:

### 1. Status Controls

- Set your status (Available, On Break, In Training, etc.)
- View your current queue assignments

### 2. Call Controls

- Make outbound calls
- Answer incoming calls
- Transfer, hold, or end calls
- Access the softphone dialpad

### 3. Client Information Panel

- View caller information
- Access client history
- Create new client records
- Document the current interaction

### 4. Knowledge Base

- Access scripts and procedures
- Find resources to assist clients
- View announcements and updates

## Using the Softphone

The MHU Helpline system includes a WebRTC-based softphone for handling calls:

### Installation

#### Windows

1. Download the softphone installer from the agent portal
2. Run the installer and follow the prompts
3. Launch the application and log in with your agent credentials

#### iOS

1. Download the MHU Helpline app from the App Store
2. Open the app and log in with your agent credentials

### Basic Call Handling

- **Answering Calls**: Click the green answer button when a call arrives
- **Making Calls**: Enter the number in the dialpad and click the call button
- **Ending Calls**: Click the red end call button
- **Transferring Calls**: Click the transfer button, select a destination, and complete the transfer

## Creating Client Sessions

For each client interaction, you'll need to create or update a client session:

1. Verify the client's identity
2. Open a new session or locate an existing one
3. Document the reason for contact
4. Record relevant details during the conversation
5. Complete any required forms
6. Add notes and follow-up actions
7. Save the session

# Mayday CRM Documentation

Welcome to the official documentation for the Mayday Helpline CRM and Call Center System - a comprehensive omnichannel platform designed to streamline mental health helpline operations, manage client interactions, and improve service delivery.

## About Mayday Helpline

Mayday Helpline is a complete omnichannel call center solution built on Asterisk 20.11.1 and Chan PJSIP. The system combines powerful telephony features with customer relationship management capabilities specifically designed for Mental Health Uganda's needs. Our platform enables support agents to create and manage client sessions while providing supervisors with robust monitoring and reporting tools.

## System Architecture

The Mayday Helpline system is built on a modern technology stack:

- **Telephony Core**: Asterisk 20.11.1 with Chan PJSIP for reliable call handling
- **Real-time Data**: Sequelize for call data and real-time operations
- **Data Storage**: MongoDB for the data tool and client session management
- **Client Interface**: WebRTC-based softphone with cross-platform support (Windows and iOS)
- **Security**: Role-based access control with granular permissions

## Key Features

### Call Center Capabilities

- **Advanced Call Management**: Handle inbound and outbound calls with features like call queuing, routing, and IVR
- **Multi-platform Softphone**: WebRTC-based softphone supporting Windows and iOS with appbar functionality for Windows
- **Audio Quality**: Built-in noise cancellation codecs for clear communication
- **Call Recording**: Secure recording and storage of calls for quality assurance and training

### Administration

- **Privileged Access Control**: Role-based permissions system for secure access management
- **Agent Management**: Create, configure, and monitor agent accounts and performance
- **Queue Configuration**: Set up and optimize call queues for different departments or services
- **IVR Design**: Visual IVR builder for creating custom call flows
- **Trunk Configuration**: Manage connections to telecom providers

### Client Management

- **Client Session Creation**: Tools for agents to document and manage mental health client interactions
- **Case Management**: Track client history and follow-up requirements
- **Data Collection**: Structured forms for gathering relevant client information

### Reporting and Analytics

- **Real-time Dashboards**: Monitor current call center activity and performance
- **Historical Reporting**: Generate reports on call metrics, agent performance, and client interactions
- **Custom Analytics**: Create tailored reports for specific operational needs
